# Ben's Bargains Clone

A faithful, fully interactive static clone of https://bensbargains.com/

## Features
- Clean, price-focused design with blue/teal accents (different from the orange Slickdeals clone)
- Prominent search + live filtering
- Deal cards with price, discount, store, score, and votes
- Click any deal to open detailed modal with "View Deal" button
- Upvoting (live updating)
- Category filters (Tech, Home, Fashion, Gaming, Auto)
- Store quick filters
- Tabs: All Hot Deals / Biggest Drops / New Today
- "Submit a Deal" form that adds new deals dynamically to the list
- Responsive (mobile friendly)
- 12 realistic seeded deals
- Tailwind via CDN + Font Awesome

## Local Preview
Just open `index.html` in your browser. No build step or server required.

## Deployment (GitHub Pages - Different URL)
This repo was created as `bensbargains-clone` so the Pages URL is separate from the Slickdeals clone.

1. The code has already been pushed to: https://github.com/frederickpsu/bensbargains-clone
2. Go to the repo → **Settings** → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main**
5. Folder: **/ (root)**
6. Save
7. Wait 1-5 minutes
8. Your site will be live at: **https://frederickpsu.github.io/bensbargains-clone/**
